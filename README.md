# 🛒 ShopEase – E-Commerce Frontend

A responsive e-commerce frontend application built using HTML, CSS, and JavaScript.
The project focuses on implementing core shopping features like product browsing, filtering, and cart management on the client side.

---

## ✨ Features

* Product listing with multiple categories
* Search and category-based filtering
* Add to cart with quantity management
* Slide-in cart panel
* Total price calculation
* Cart persistence using localStorage
* Responsive layout for different screen sizes

---

## 🛍️ Categories

* **Fashion**

  * Men (shirts, t-shirts, jeans)
  * Women (kurtis, sarees, dresses)
  * Kids (boys & girls wear)

* **Accessories**

  * Sunglasses
  * Watches
  * Bags

* **Ornaments**

  * Artificial jewelry (necklace, earrings, bangles)

* **Household Items**

  * Kitchen essentials
  * Home decor
  * Daily-use items

## ▶️ Running the Project Locally

1. Clone the repository:

   ```
   git clone https://github.com/your-username/E-Commerce-WebApp.git
   ```

2. Navigate into the project folder:

   ```
   cd E-Commerce-WebApp
   ```

3. Open the project:

   * Open `index.html` directly in your browser
     **or**
   * Open the folder in VS Code and use Live Server

---

## 📌 Notes

* This is a frontend-only project (no backend integration)
* Cart data is stored in the browser using localStorage
* Product data is currently static and can be extended
